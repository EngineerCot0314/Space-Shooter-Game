### Objective

- Destroy meteors using lasers  
- Avoid colliding with meteors  
- Survive as long as possible  
- The game ends when your ship hits a meteor  

---

### How to Run

1. Download or clone this repository  
2. Open a terminal in the project folder  
3. Run:

python main.py

(Replace main.py if your file name is different)

### Features

- Smooth player movement  
- Laser shooting with cooldown  
- Animated explosions  
- Random meteor spawning  
- Background music and sound effects  
- Score system based on survival time
